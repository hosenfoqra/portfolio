Smart School App

This project is a school management system designed to enhance communication and collaboration between parents, teachers, and students. The system aims to provide real-time updates on students' academic and social progress, bridging the gap between home and school. It offers features such as attendance tracking, grade reporting, exam schedules, study materials sharing, and a calendar with holidays and school events. It emphasizes user-friendly design and accessibility across multiple platforms.

  • Frontend Development: React Native (Expo): For building a cross-platform mobile application with a responsive and user-friendly interface. React Native Elements: For pre-built components to streamline the UI/UX design process.
  • Backend Development: Node.js (Express.js): To create a robust backend for handling application logic and APIs.
  • Database Management: PostgreSQL: For efficient storage, management, and querying of data related to users, attendance, grades, and communication logs.
  • Figma: For creating wireframes, prototypes, and a visually appealing design of the application.
  • Development Tools: Git and GitHub: For version control and collaborative development. NPM and Yarn: For managing JavaScript dependencies and libraries. Visual Studio: As a primary development environment.
  • Project Management: Jira: To manage tasks, track progress, and plan sprints using Agile methodologies.

This project not only addresses current gaps in school communication tools but also serves as a practical learning experience in modern software development and system integration.
